C# project that queries an API by obtaining and handling data about Star Wars characters:

A project written in .NET 8.0, querying the API (https://swapi.dev/ where read-only data is exposed) obtains information regarding Star Wars characters and then thanks to the Spectre Console library shows a menu from which one can obtain: the list of characters, average their height and weight, and search by name.

Gitter Writer Android App:

A simple type app, where you can choose beetwen three languages, Clicking on language it will open a new Activity where a bunch of random words are generated and you will find a "START" button, if you feel ready you can press it and a count down will start along with an editable text, type the words and when the countdown ends check how many words have you have written correctly. After that press "GO BACK" and you can restart everything.
